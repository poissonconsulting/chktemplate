# Single Demo Template

Sample template for one table.

## Usage

``` r
demo_template_tag
```

## Format

A data.frame with columns:

- name:

  Row description

- date:

  Date the fish was caught

- tag_1:

  The number from the first tag

- tag_2:

  The number from the second tag

- harvested:

  Either yes or no for if the fish was harvested

- comment:

  Comments about the capture
